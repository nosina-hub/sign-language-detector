# 🤟 SignBridge - ASL Translator

An AI-powered Sign Language Detector that translates hand signs into text and speech in real-time.

## 🚀 Features
- **Real-time Detection:** Uses MediaPipe and OpenCV to track hand movements.
- **Voice Output:** Automatically speaks the translated signs using the Web Speech API.
- **Interactive UI:** Built with Flask and a modern, responsive dashboard.

## 🛠️ Tech Stack
- **Backend:** Python (Flask)
- **AI/ML:** MediaPipe, Scikit-learn, NumPy
- **Frontend:** HTML5, CSS3, JavaScript
- **Deployment:** Render

## 📦 Installation
1. Clone the repo: `git clone https://github.com/nosina-hub/sign-language-detector.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Run locally: `python app.py`
